Math Proficiency Test Prep Webstite

As part of my leadership project in PED 3151 while attending uOttawa for my BEd., I created an interactive open-source Math Proficiency Test (MPT) Prep website . The key features of this site include a Duolingo-style preperation game (XP tracking, and streaks), as well as a collection of practice tests, and a visual summary of the MPT framework document.
